RapidHealth
SF Hacks Award For Best Use of Notivize
Entered a virtual hackathon competition with 700+ contestants and collaborated with 4 other team members. With a time limit of three days, this was the result of our work. RapidHealth sends you SMS and email notifications using Notivize, a notification API, and alerts you whenever the number of COVID-19 cases increases or decreases in your area. All you have to do is enter your contact information. Our project won an honor award for best use of Notivize's API in March, 2021.
【wavyboy】synthesizer
A music producer should know how a synthesizer works. So to elevate my own understanding, I decided to build one myself. With the Tone.js framework, I designed and constructed my own.【wavyboy】features EQ gain knobs, volume and reverb/delay sliders, and pitch-shifting buttons for maximum sound customizability.
Weather Wherever
Checking the weather is a very useful aspect in daily life. Maybe you might need to check the weather before you hang your laundry outside. Low and behold, my weather app designed for mobile devices uses geolocation to detect user location and display the temperature. As an added touch, every weather transitions smoothly into different background themes for a more pleasant experience. You can also search for any location for the temperature using Google Places API.
Adulting.io
Completing tasks are difficult to accomplish unless you can keep track of everything you need to complete. My easy to use to-do list app includes the ability to add, delete, and clear all new tasks using JSON, saving them in cache when the application is closed.
Music Website Player
A music website player that I built using JS, HTML/CSS, JQuery, and HTML5 Player. This features different background themes to each song selection. All vocals and music production are done with Logic Pro X and yours truly.
